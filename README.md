## 🛠️ Getting Started

1. **Clone this repo**
   ```bash
   git clone https://github.com/TheRealSaiTama/LinkTree.git
   cd LinkTree
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Launch in dev mode**
   ```bash
   npm run dev
   ```
4. **Open your browser** at `http://localhost:5173` and witness the magic.
